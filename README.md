# Paradise Nursery Shopping Application

## Project Description

Paradise Nursery is an online shopping application that allows users to browse a selection of house plants and add them to a shopping cart.

## Features

- Landing page with a Get Started button
- Navigation bar
- Product listing page
- Two plant categories:
  - Aromatic Plants
  - Medicinal Plants
- Add plants to cart
- Increase and decrease quantity
- Delete items from cart
- Shopping cart with total cost
- Continue Shopping button
- Checkout button

## Technologies Used

- React
- React Router DOM
- JavaScript
- HTML
- CSS
